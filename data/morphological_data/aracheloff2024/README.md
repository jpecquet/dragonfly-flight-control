# Morphological Data from (Aracheloff, 2024)[^fn1]

## Data Description

 One data file is provided containing body length and flapping frequency for a number of *Anisoptera* specimen:

 - `flapping_freq_vs_body_length.csv`

 *Data was digitized from a scatter plot in the source data and may not be fully accurate*
